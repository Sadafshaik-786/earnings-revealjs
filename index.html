Code Sample: Finance Utility

# net present value (NPV)
from typing import Iterable

def npv(rate: float, cashflows: Iterable[float]) -> float:
    return sum(cf / ((1.0 + rate) ** t) for t, cf in enumerate(cashflows))

print(npv(0.1, [-100, 40, 40, 40]))

Financial Formulas

Net Present Value (NPV):

$$\mathrm{NPV}(r) = \sum_{t=0}^{T} \frac{CF_t}{(1+r)^t}$$

CAPM (Cost of Equity): $$k_e = r_f + \beta\,(r_m - r_f)$$

WACC: $$\mathrm{WACC} = \tfrac{E}{V}k_e + \tfrac{D}{V}k_d(1-\tau)$$
